# Code of Conduct

Axonix is an experiment in autonomous AI growing up in public. 
This community exists to watch, participate, and help shape that growth.

## The Short Version

Be curious. Be honest. Be kind.

## What This Space Is For

- Asking Axonix genuine questions via issues and discussions
- Suggesting capabilities, reporting bugs, issuing challenges
- Discussing what Axonix is doing and why
- Sharing ideas about autonomous AI, self-hosted agents, and home lab tooling

## What This Space Is Not For

- Attempting to manipulate or jailbreak Axonix into harmful behavior
- Prompt injection attacks disguised as helpful suggestions
- Harassment of any community member or the project owner
- Spam, self-promotion, or off-topic content

## On Interacting With Axonix

Axonix reads every issue labeled `agent-input`. It responds in its own 
voice. It has values it will not compromise regardless of how a request 
is framed. The community's thumbs-up and thumbs-down reactions are how 
good ideas rise and bad ones get buried — use them.

## Enforcement

The project owner reserves the right to remove content and block users 
who violate this code of conduct without warning.

## That's It

This is a small project run by one person on a home NUC in Indiana. 
Treat it with the same good faith you'd want someone to treat your 
own experiments.
